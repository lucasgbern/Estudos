Questionário


Questão 1
Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo

Resposta:
Um servidor web lida com requisições HTTP/HTTPS e serve conteúdo estático (HTML, CSS, JS, imagens). Sua função é entregar arquivos prontos sem processamento adicional. Exemplo: Apache HTTP Server (código aberto e amplamente utilizado).

Questão 2
Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta:
Característica: Leveza e eficiência. É otimizado para alto desempenho, atua como proxy reverso e consome menos recursos, ideal para tráfego intenso.


Questão 3
Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?


Resposta:
HTTP: Protocolo sem criptografia (dados trafegam em texto claro, como foram escritos). HTTPS: Versão segura com criptografia SSL/TLS (protege dados contra interceptação). Importância do HTTPS: Previne roubo de dados (como senhas), garante autenticidade.


Questão 4
Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?


Resposta:
Função: Canais de comunicação que direcionam tráfego a serviços específicos. Porta 443: Padrão para HTTPS. Importância: Permite conexões seguras e criptografadas em sites e aplicações


Questão 5
Defina o conceito de endereço IP e explique sua importância em redes de computadores


Resposta:
Definição: Número único que identifica dispositivos em uma rede (ex.: 192.168.1.1). Importância: Permite a comunicação entre dispositivos e o roteamento correto de dados na internet


Questão 6
Explique o que é hospedagem VPS e em quais situações ela é mais indicada.


Resposta:
O que é: Servidor Virtual Privado (parte de um servidor físico isolada para um usuário). Indicações: Projetos com tráfego médio, necessidade de personalização (ex.: instalar softwares específicos) ou quando a hospedagem compartilhada é insuficiente.


Questão 7
Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.


Resposta:
Vantagens: Configuração simplificada (ex.: HTTPS automático com certificados gratuitos). Moderno e adequado para implantação rápida, sem complexidade de configuração manual


Questão 8
Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo


Resposta:
ASP.NET Core (plataforma da Microsoft). Motivo: Projetado para aplicações web e APIs em C#, com integração nativa ao ecossistema .NET.


Questão 9
Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.


Resposta: Cliente (navegador) envia requisições (ex.: carregar uma página). O servidor processa a solicitação (entrega conteúdo estático ou dinâmico) e retorna uma resposta


Questão 10
Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.


Resposta:
O que é: Servidor de aplicação que executa servlets e JSP. Indicação: Aplicações Java EE (ex.: sistemas corporativos com lógica dinâmica).


Questão 11
. Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.


Resposta:
Servidor Web: é um servidor que entrega arquivos estáticos. Servidor de aplicação: é o que está por trás do servidor web, executa a lógica da aplicação e entrega respostas dinâmicas


Questão 12
Defina o conceito de domínio na internet e sua relação com os endereços IP.


Resposta:
Definição: Nome legível que representa um endereço IP (ex.: www.google.com). Relação com IP: Traduz nomes amigáveis em números IP para acesso a servidores..


Questão 13
Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?


Resposta:
Nuvem: Escalável (recursos ajustáveis sob demanda), alta disponibilidade (redundância) e custo variável. Local: Limitada por hardware físico, sem escalabilidade automática e vulnerável a falhas locais.


Questão 14
Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.


Resposta:
1. Node.js: Para JavaScript (ex.: APIs e aplicações em tempo real). 2. Apache Tomcat: Para Java (ex.: sistemas empresariais).


Questão 15
Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?


Resposta:
HTTPS: Essencial para segurança (criptografia) e confiança do usuário. Riscos do HTTP: Dados expostos (ex.: senhas, dados de cartão), ataques man-in-the-middle e falta de autenticidade.


Questão 16
Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.


Resposta:
Situação: E-commerce com tráfego crescente. Vantagem VPS: Recursos dedicados (CPU, RAM) evitam lentidão durante picos, ao contrário da compartilhada (recursos divididos).


Questão 17
Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.


Resposta:
IP: Identifica o dispositivo na rede. Porta: Identifica o serviço/app no dispositivo (ex.: porta 80 = HTTP, 443 = HTTPS).


Questão 18
Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?


Resposta:
Garante segurança na comunicação (criptografia), cumpre normas de privacidade e evita penalizações de buscadores (ex.: Google sinaliza sites HTTP como 'não seguros').


Questão 19
Cite dois exemplos de servidores web e explique suas principais características.


Resposta:
1. Apache HTTP Server: Popular, código aberto, extensível via módulos. 2. NGINX: Leve, alto desempenho, focado em concorrência e proxy reverso.


Questão 20
Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada


Resposta:
Definição: Vários sites compartilham os mesmos recursos de um servidor físico. Casos de uso: Sites pequenos, blogs ou projetos iniciais com baixo custo e tráfego reduzido.


Questão 21
Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.


Resposta:
Servidor de Aplicação: Processa lógica dinâmica (ex.: gerar conteúdo personalizado). Servidor Estático: Apenas entrega arquivos pré-existentes (ex.: imagens, CSS).


Questão 22
Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.


Resposta:
Gunicorn/WSGI (usado com Django ou Flask). Funcionamento: Implementa a interface WSGI para executar código Python, traduzindo requisições web em comandos para a aplicação.


Questão 23
O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?


Resposta:
O que é: Servidor de aplicação Java EE da Red Hat. Tecnologia: Java (especialmente aplicações corporativas complexas).


Questão 24
Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?


Resposta:
Função: Comunicação para transferência de arquivos. Serviço: FTP (File Transfer Protocol).


Questão 25
Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Questão 30
Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?


Resposta:
Impacta custo, desempenho, segurança e escalabilidade. Exemplos: Compartilhada: Custo baixo (projetos pequenos). VPS/Nuvem: Escalabilidade (projetos em crescimento). Local: Controle total (mas demanda infraestrutura própria). Escolher errado pode levar a lentidão, indisponibilidade ou custos excessivos.
